#### Установка
1. Скачать репозиторий:
```git clone https://github.com/nik-di/pacman-preloader.git```
2. Установить npm-зависимости:
```npm i```
3. Произвести сборку проекта:
```npm run build```
4. Запустить проект на локальном сервере:
```npm run start```
